Build Data Collection Forms in wordpress with various type of fields like File, Image upload, Date, Select, radio, checkbox and text fields.

== Description ==

Build any type of Data Collection form with various type of input fields and collect data from any page of post in the wordpress posts.

WATCH DEMO: <a href="https://www.youtube.com/watch?v=9arMo2Pg7i0">https://www.youtube.com/watch?v=9arMo2Pg7i0</a>

1. Data Collection form Build form and display in via short code or through php function.
2. Captcha enabled Data Collection form to avoid spammers.
3. Token based Data Collection form to avoid spammers.
4. Date scheduling for Data Collection form.
5. Display Data Collection form any where with short code or via php  function.
6. Stores form data in any type of post type. It creates meta field automatically if you have not created.
7. Data Collection form supports image upload and file upload.
8. Data Collection form supports date field with jquery date picker.
9. Data Collection form is full customizable form with css and html.
10. User can Enable/Disable Data Collection form with one click.


== Installation ==

1. Upload `data-collection-form` to `wp-content/plugins` directory
2. Activate plugin through the 'Plugins' menu in WordPress
